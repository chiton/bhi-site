## Instructions

When generating a page:

1. Use Next.js App Router (`app/` directory)
2. Default to Server Component unless interactivity is needed
3. Include:
   - Page component
   - Metadata export
4. Use modern conventions (async components if needed)
5. Keep layout clean and minimal

## Output format

- File path
- Full code
